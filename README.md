# 王中王工具箱

一款功能强大的实用工具集合，采用现代化界面设计，提供文本处理、图片编辑、网络诊断、加密解密等多种实用工具。

![技术栈](https://img.shields.io/badge/技术栈-Electron+React-purple)

## 功能特性

### 文本工具
- **文本格式化** - 自动格式化 JSON、XML 等文本
- **正则表达式测试** - 实时测试和调试正则表达式
- **编码转换** - 支持多种字符编码相互转换

### 图片工具
- **图片压缩** - 智能压缩图片，保持画质的同时减小体积
- **格式转换** - 支持 PNG、JPG、WebP 等格式互转
- **图片调整** - 修改尺寸、裁剪、旋转等操作
- **图片信息** - 查看图片 EXIF 等元数据

### 网络工具
- **IP 查询** - 查询本机 IP 和公网 IP 信息
- **端口扫描** - 扫描目标主机开放端口
- **DNS 测试** - 查询域名 DNS 记录（A、AAAA、MX、NS、CNAME、TXT、SOA）
- **Ping 测试** - 测试网络连通性和延迟
- **Whois 查询** - 查询域名注册信息
- **HTTP 检查** - 检测网站响应状态和时间

### 加密工具
- **哈希生成** - 支持 MD5、SHA1、SHA256、SHA384、SHA512
- **Base64 编解码** - 字符串 Base64 编码和解码
- **URL 编解码** - URL 字符串编码和解码
- **UUID 生成** - 批量生成 UUID/GUID 唯一标识符
- **二维码生成** - 将文本或 URL 生成二维码图片
- **JWT 解析** - 解码和查看 JWT Token 内容

## 技术栈

- **Electron** - 跨平台桌面应用框架
- **React** - 用户界面构建
- **Vite** - 构建工具
- **Node.js** - 后端功能支持

## 安装使用

### 下载安装
1. 下载最新版本的安装包
2. 双击运行安装程序
3. 按照向导完成安装

### 免安装版本
也可以下载免安装版本，解压后直接运行 `王中王工具箱.exe`
